{ utveckling ; }

Vi kan hjälpa er genom hela utvecklingskedjan från idé till färdig produkt. Eller har ni kanske redan ett system men önskar modifiera eller modernisera det? Oavsett vilket så besitter vi med vårt breda kontaktnär och vår mångåriga erfarenhet från it-branchen rätt kompetens. För att vara säkra på att ni i slutändan verkligen får det system ni efterfrågar arbetar vi agilt i korta utvecklingscykler med regelbundet återkommande avstämningsmöten med er, speciellt då vi vet att systemutveckling är en levande process under vilken nya ideer och tankar ständigt kommer upp.



Måsvinge äger aldrig koden utan ni äger den fullt ut från dag 1. Naturligtvis tar vi oss an driften om så önskas men vill ni hellre själva hålla i spakarna så ser vi inga problem med det.

{ underhåll && förvaltning ; }

Har ni ett excelark som har gått över styr, en gammal gammal access applikation eller en webb applikation som börjar se lite trött och sliten ut? Då kan vi hjälpa er. Vi börjar med att analysera applikationen för att kunna bedömma vad som är långsiktig bäst; fortsatt underhåll eller är det helt enkelt billigare att bygga nytt. Vi vet att många gamla organisationskritiska system är krångligt skriva, ofta påbyggda lite för många gångar och kanske finns inte konstruktören längre tillgänglig. Men vi gillar den problemlösning och det detektivarbete som ofta kommer med övertagandet av förvaltningen av äldre system. Vi analyserar ert system och ger er en offert på hur vi vill gå vidare kostnadsfritt.

{ drift ; }

Vi driftar idag de mesta av system i Microsoft Azure molntjänst eller Amazon cloud services lite beroende på vilket system, krav och prisbild. Vår erfarenhet visar att molntjänsterna nästan alltid blir den bästa långsiktiga lösningen med avseende på såväl prestanda, driftsäkerhet & pris. Har ni däremot speciella lagkrav eller andra särskilda krav så samarbetar vi även med ett svensk driftföretag i norra Sverige där vi har våra fysiska maskiner.
